# Pohja Ohjelmointi 1 -kurssin harjoitustöiden Git-varastoille.

Tämä on pohja Ohjelmointi 1 -kurssin harjoitustöiden Git-varastoille. 

 1. _Forkkaa_ tämä repon omaa harjoitustyötäsi varten. `fork`-toiminto löytyy sivun yläreunasta.
 2. Muokkaa <https://gitlab.jyu.fi>-palvelussa oman projektisi asetuksia kohdassa Settings. Vaihda _ainakin_ näkyvyysasetus kohtaan public, jotta ohjaaja pääsee tarkastelemaan työtäsi.
 3. Muokkaa suunnitelma ja pelikoodia Git-työkaluja käyttäen. Tiedostojen lisääminen tai muokkaaminen GitLab-palvelun kautta (selaimessa) Upload-toimintoa käyttäen **on ehdottomasti kielletty**. 
 1. Ennen tiedostojen lähettämistä etävarastoon kannattaa tarkistaa oman kansiosi kelvollisuuden Git Bashissa (Windows, Linux) tai Terminalissa (Mac) komennolla `bash check.sh`. Komento näyttää, onko juurikansiossasi sinne kuulumattomia tiedostoja. Jos tiedostoja löytyy, poista ne tai siirrä ne toiseen kansioon. Komento `bash check.sh` ei poista tiedostoja, vaan ainoastaan ilmoittaa niistä.
 
Lue huolellisesti Git-versiohallinnan tarkemmat käyttöohjeet TIMistä: <https://tim.jyu.fi/view/kurssit/tie/ohj1/harjoitustyo/git>.
